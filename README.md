# Homebrew Tap

Personal Homebrew tap for @licht1stein.

## Installation

```bash
brew tap licht1stein/tap
brew install sanskrit-upaya
```

Or install directly:

```bash
brew install licht1stein/tap/sanskrit-upaya
```

After installation, launch the app:

```bash
sanskrit-upaya
```
